# Decision Companion System

## Overview
The **Decision Companion System** is a tool designed to help users make better decisions by evaluating multiple options against weighted criteria. The system provides a ranked recommendation and an explanation of why a particular option is preferred.

This project focuses on a **transparent, explainable approach**, using **Multi-Criteria Decision Making (MCDM)** methods such as Weighted Scoring and Analytic Hierarchy Process (AHP), rather than relying entirely on AI models.
